# Concurso Agrinho 2026 - Programação Front-End (Subcategoria 3)

## 📌 Informações do Projeto
- **Tema:** Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.
- **Categoria:** Programação Front-End (Ensino Médio).
- **Tag Obrigatória:** `agrinho` (adicionada aos Topics do Repositório).

## 🎯 Objetivo do Projeto
Este site foi projetado para conscientizar os estudantes e usuários sobre a importância de alinhar o desenvolvimento econômico do Agronegócio com a preservação ecológica. Através de uma interface limpa e um jogo interativo estilo *Snake* adaptado (Simulador de Colheita Ecológica), o usuário compreende de forma prática que coletar sementes certificadas aumenta o rendimento, enquanto avançar sobre áreas de degradação ambiental interrompe a sustentabilidade e encerra a atividade.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica do conteúdo.
- **CSS3:** Estilização moderna, responsividade completa via *Media Queries* e suporte nativo a Temas (Light/Dark).
- **JavaScript (ES6):** Manipulação dinâmica de eventos, controle de fluxo e renderização lógica baseada em Canvas.

## 🕹️ Instruções de Uso
1. Insira o nome do Produtor no formulário inicial para receber as saudações customizadas.
2. Utilize as **setas direcionais do teclado** (Cima, Baixo, Esquerda, Direita) para movimentar a colheitadeira no tabuleiro.
3. Colete o máximo de **Sementes Certificadas (Quadrados Verdes)** para acumular 10 pontos por item.
4. Evite colidir com as bordas, contra o próprio corpo ou com as **Áreas de Degradação (Quadrados Vermelhos)** para não sofrer *Game Over*.
5. Use os botões superiores para ajustar preferências visuais como Modo Escuro e Aumento de Fonte.
